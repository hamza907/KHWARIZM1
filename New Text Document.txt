بما إن التسجيل بإيميل الأدمن بقى ممنوع من الفورم العام، مفيش طريقة عامة تنشئ بيها حساب الأدمن الحقيقي. علشان تعمله لنفسك بس (مش حد تاني هيقدر يعمل كده)، افتح الموقع في المتصفح، دوس F12 لفتح Console، واكتب الكود ده مرة واحدة بس (غيّر الإيميل والباسورد براحتك):


async function createAdmin(email, password, name) {
  const data = new TextEncoder().encode(password);
  const buf = await crypto.subtle.digest("SHA-256", data);
  const passwordHash = Array.from(new Uint8Array(buf)).map(b=>b.toString(16).padStart(2,"0")).join("");
  const users = JSON.parse(localStorage.getItem("users")) || [];
  users.push({ name, email, passwordHash, role: "admin", createdAt: new Date().toISOString() });
  localStorage.setItem("users", JSON.stringify(users));
  console.log("تم إنشاء حساب الأدمن ✅");
}
createAdmin("admin@khwarizm.com", "ضع_باسورد_قوي_هنا", "حمزة");
